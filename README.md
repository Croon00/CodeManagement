# CodeManagement

## 코드 관리 시스템

### 백엔드
- **Spring Boot**: 애플리케이션 개발 프레임워크  
- **Spring Security**: 인증 및 권한 관리  
- **Spring Data JPA**: 데이터베이스 ORM  
- **JWT (JSON Web Token)**: 인증 토큰 관리  
- **MySQL**: 관계형 데이터베이스  
- **SpringDoc OpenAPI**: API 문서 자동 생성  

### 프론트엔드
- **React**: 사용자 인터페이스 개발 라이브러리  
- **MUI (Material-UI)**: React UI 컴포넌트 라이브러리  
- **Axios**: HTTP 요청 처리  
- **Recoil**: 상태 관리 라이브러리
